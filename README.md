《洛基第二季》（Loki Season 2）
是由Disney+出品，艾瑞克·马汀等人编剧，贾斯汀·本森、艾伦·穆尔黑德、丹·德里伍、凯西拉·法拉尼等人执导，
汤姆·希德勒斯顿、索菲娅·迪·马蒂诺、欧文·威尔逊、乌米·马萨库、尤金·科德罗等主演的悬疑犯罪剧。 
该剧改编自漫威漫画，讲述了洛基被时间管理局抓住之后，联手特工莫比乌斯，猎人B-15、OB特工以及其他几个宇宙中的洛基，发现时间管理局背后的真相：一切都是反派“康”的阴谋。 
该剧于2023年10月5日在Disney+播出，首播前三天全球观看次数为1090万次，在2023年10月9日至15日一周，《洛基第二季》成为美国所有平台上收视率最高的电视连续剧。
